Available Variables

You can use these variables in your response templates to personalize messages.

Variable Description Example
{name} User's first name Hello, {name}!
{full_name} User's full name Thank you, {full_name}
{username} User's Telegram username I'll contact @{username}
{date} Current date Today is {date}
{time} Current time Message received at {time}

Formatting Options

Telegram supports these formatting options in messages:

Format Syntax Example
Bold **text** Important notice
Italic *text* Special emphasis
Underline __text__ Underlined text
Strikethrough ~~text~~ Crossed out
Code `code` monospace text
